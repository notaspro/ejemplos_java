
Validar instalacion de NodeJS

Validar El Node Package Manager

Instalacion de TypeScript con el  Node Package Manager 
(en linea de comandos modo administrador o con sudo)
npm install -g typescript


Validar instalacion de TypeScript
tsc --version

Instalar el Gestor de Angular con el  Node Package Manager 
npm install -g @angular/cli

Validar la instalacion del Gestor de Angular (Angular/CLI)
ng --version

Instalar IONIC, sirve para crear aplicaciones moviles
(https://ionicframework.com/)
npm install -g ionic
